RB Cloud Web SDK

Introduction

RB Cloud Web SDK는 SK Planet에서 만든 3D 라이브러리를 외부의 개발자가 쉽게 이용할 수 있도록 SDK 형태로 공개하는 기술 입니다.

RB Cloud Web SDK는 WebGL과 Vue를 기반으로 하는 3D 기술로 구현되어있으며 본 가이드에서는 홈/GNB/편성표 등에 대한 component 들의 사용법를 알 수 있습니다.



Web SDK의 구성 요소는 아래와 같습니다.
구성요소설명
라이브러리코드 패키지
API서비스를 제공하기 위한 인터페이스
IDE지원되는 코드 에디터는 VC code
GuideSDK 사용 가이드
Components제공하는 컴포넌트들과 예제 코드
