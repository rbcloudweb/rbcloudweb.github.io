라이브러리 설치

NPM 패키지로 제공되며 npm install로 라이브러리를 다운로드 받을 수 있습니다.

npm install vcs-sdk

라이브러리에서 제공되는 UI 효과 사용

라이브러리에서 제공되는 효과는 2가지 방식으로 사용 할 수 있습니다.

  1. 이벤트를 이용한 효과 개발
cs.excuteEventListener(cs.VCS_SDK_BACKGROUND, imagePath);
  1. html tag를 이용한 효과 개발
<template>
    <vcs-sdk-background class="bg" :options="options"></vcs-sdk-background>
</template>

이미지 전환 효과 - Background Image Effects

포커스된 콘텐츠의 배경 이미지 전환 효과를 제공합니다.

GIF 이미지 추가 예정

  • eventBus를 사용하여 이미지 주소를 set / remove 하는 방법
  • 함수를 사용하여 이미지 주소를 set / remove 하는 방법

Example Usage

<template>
  <rbcloud-background class="bg" :options="options"></rbcloud-background>
</template>

<script setup>
import Vue from "vue";
import { watch, onBeforeUnmount } from Vue;

const eventBus = new Vue();

const props = defineProps({
  imgSrc: String
});

const options = {
  slideType = 'crossover';
};

watch(
    () => props.imgSrc,
    (src) => {
          if (src) {
              eventBus.$emit(RGC_EVENT.SET_BACKGROUND, {
                  src
              });
          }
    }
);

onBeforeUnmount(() => {
    eventBus.$emit(RGC_EVENT.REMOVE_BACKGROUND
});
</script>
