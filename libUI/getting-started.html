RGCloudBackground

WebGL을 사용하여 Background 이미지를 변경할 때 다양한 효과를 사용하도록 지원

RGCloudBackground는 두가지 방법을 지원합니다.

  • eventBus를 사용하여 이미지 주소를 set / remove 하는 방법
  • 함수를 사용하여 이미지 주소를 set / remove 하는 방법

Example Usage

<template>
  <rbcloud-background class="bg" :options="options"></div>
</template>

<script setup>
import Vue from "vue";
import { watch, onBeforeUnmount } from Vue;

const eventBus = new Vue();

const props = defineProps({
  imgSrc: String
});

const options = {
  slideType = 'crossover';
};

watch(
    () => props.imgSrc,
    (src) => {
          if (src) {
              eventBus.$emit(RGC_EVENT.SET_BACKGROUND, {
                  src
              });
          }
    }
);

onBeforeUnmount(() => {
    eventBus.$emit(RGC_EVENT.REMOVE_BACKGROUND
});
</script>
