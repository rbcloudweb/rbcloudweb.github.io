<rbcloud-block>

An abstract Block component.

Example Usage

<template>
  <CarouselBlock
    :poster-path="data.poster"
    :transform="data.transform"
  />
</template>

<script setup>

import CarouselBlock from './CarouselBlock'
import { ref } from 'vue'

const data = ref({
  poster: 'url(https://stimage.hanafostv.com:8443/thumbnails/iip/480_720_10/movie/05/MM1000998905/MM1000998905_TVPH_20240312104910962_01.jpg)',
  transform: 'rotateY(0deg) translateZ(2379px) rotate3d(1, 0, 0, 0deg)'
})

</script>

  <style
    scoped
    lang="scss"
  >

  .block-poster {
    position: absolute;
    border: 2px solid black;
    text-align: center;
    //width: 480px;
    //height: 720px;
    transform: translateZ(-288px);
    box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06);

    &.--focused {
      transition: transform 0.5s;
      zoom: 1.02;

      border: 17px solid #7387ffff;
      border-radius: 80px / 60px;
    }

    &.--blur {
      transition: transform 0.4s;
      opacity: 0.8;
    }
  }

  </style>

Props

  • posterPath : string
    블록 포스터 이미지.
  • transform : string
    블록 포스터 위치 (3d).
