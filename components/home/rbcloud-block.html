<rbcloud-block>

An abstract Block component.

Example Usage

<template>
  <div class="container">
    <div
      class="block-container"
    >
      <home-CarouselBlock
        :epsd-id="data.epsdId"
        :poster-path="data.poster"
        :transform="data.transform"
        :height="data.height"
        :width="data.width"
        :selected="data.selected"
      />
    </div>
  </div>

  <aside>
    <label>height<input
      v-model.number="data.height"
      type="range"
      min="100"
      max="600"
    ></label>
    <label>width<input
      v-model.number="data.width"
      type="range"
      min="100"
      max="600"
    ></label>

    <label>Selected<input
      v-model="data.selected"
      type="checkbox"
    ></label>
  </aside>
</template>

<script setup>

import { ref } from 'vue'

const data = ref({
  epsdId: 'MM1000998905',

  transform: 'rotateY(4deg) translateZ(2000px) rotate3d(1, 0, 0, 0deg)',
  height: 720,
  width: 480
})

</script>

Props

  • epsdId : string
    블록 식벽 키.
  • posterPath : string
    블록 포스터 이미지.
  • transform : string
    블록 포스터 위치 (3d).
    Defaults to () => { return 'none' }
  • selected : boolean
    블록 포스터 선택 여부
    Defaults to () => { return false }
  • height : number
    블록 포스터 높이
  • width : number
    블록 포스터 넓이
